Slider

Para agregar un widget slider a su página, utilizar un estándar de entrada con el type = "range" atributo. La entrada de valor se utiliza para configurar la posición inicial de la manija y el valor se rellena en la entrada de texto. Especifique min y max valores de atributos para establecer el alcance del control deslizante. Si desea restringir la entrada a incrementos específicos, agregue el paso atributo. Establezca el valor del atributo para definir el valor inicial. El marco analizará estos atributos para configurar el widget slider. Ver la referencia-atributo de datos para ver todos los atributos posibles que puede añadir a los deslizadores.
Al arrastrar la manija de la barra deslizante, el marco se actualizará el valor de la entrada nativa (y viceversa) por lo que siempre están en sincronía; esto asegura que el valor se envía con el formulario.
Establezca el de atributo de la etiqueta para que coincida con el ID de la entrada para que se asocian semánticamente. Es posible ocultar accesible la etiqueta si no se desea en el diseño de la página, pero le exigimos que está presente en el marcado por razones semánticas y de accesibilidad.
El marco encontrará todos los insumos elementos con un type = "range" y automáticamente mejorarlas en un slider con una entrada de acompañante sin necesidad de aplicar un data-role atributo. Para evitar que la mejora automática de esta entrada en un control deslizante, añada data-role = "none" atributo a la entrada y se envuelven en un div con el data-role = "fieldcontain" atributo para agruparlos. En este ejemplo, el rango aceptable es de 0-100.

Interruptor de palanca del tirón Un interruptor binario "flip" es un elemento de la interfaz de usuario común en los dispositivos móviles que se utiliza para el binario de encendido / apagado o la verdadera entrada de datos / falso. Puede arrastrar la manija del tirón como una barra de desplazamiento o toque en un lado del interruptor.

La primera opción será el estilo como "on" serán decoradas Detector de estado y el segundo como el estado "apagado" para escribir sus opciones en consecuencia.
